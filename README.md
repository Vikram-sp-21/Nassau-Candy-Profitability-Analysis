# Nassau Candy Profitability Analysis

## Project Overview

This project analyzes profitability, margins, division performance, and revenue concentration for [Nassau Candy Distributor](https://www.nassaucandy.com/). 

## Tools Used

- Python
- Pandas
- Plotly
- Streamlit
- Jupyter Notebook

## Key Insights

- 4 products generate 80% of revenue.
- 4 products generate 80% of profit.
- Cost-Sales Correlation = 0.96.
- California, New York, and Texas are the largest revenue contributors.
- 16.07% of products require repricing.

## Dashboard Features

- Product Profitability Overview
- Division Performance Dashboard
- Cost vs Sales Diagnostics
- Pareto Analysis
